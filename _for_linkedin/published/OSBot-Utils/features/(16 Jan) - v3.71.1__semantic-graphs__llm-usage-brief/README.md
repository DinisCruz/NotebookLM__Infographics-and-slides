## Key Themes

- **The Four Pillars**: Taxonomy, Ontology, Graph, Projection
- **ID/Ref Duality**: Internal IDs for integrity, human-readable refs for display
- **Schema vs Logic Separation**: Schemas are pure data, business logic is separate
- **Validation Engine**: Validate graphs against ontology constraints
- **Type-Safe Throughout**: Runtime type checking on every operation
